# 🏆 Table Wars! Host Guide

Welcome to the official **Table Wars!** event management system. This guide will help you set up, manage, and run a professional tabletop competition from start to finish.

---

## 🏗️ 1. Setup Phase
The Setup screen is your control center before the audience arrives.

*   **Competition Branding**: Click the main "Table Wars!" title to customize the event name (e.g., *Summer 2026 Finals*).
*   **Team Configuration**: 
    *   Add up to 8 teams.
    *   Give each team a unique name and a "Chant" (which will be displayed during their introduction).
*   **Device Sync**: 
    *   Click **"Enable Multi-Device Sync"** to generate a 6-character session code.
    *   Give this code to Team Captains. They can join at `yoursite.com/join`.

---

## 🎮 2. Running the Event
Once you click **"Start Competition"**, you enter the Host Panel.

### The 5 Rounds
The system is pre-configured for a classic 5-round battle:
1.  **Table Trivia**: Uses the built-in buzzer system.
2.  **Food Relay**: Manual score entry.
3.  **Table Building**: Manual score entry.
4.  **Blind Taste Test**: Specialized sensory round controls.
5.  **Grand Finale**: High-stakes buzzer round.

### Projection Control
Use the footer buttons to control what the audience sees on the big screen:
*   **LOGO**: Standby screen with your custom competition branding.
*   **SCOREBOARD**: The live leaderboard.
*   **BLACK**: Instantly hide all content (Blackout).
*   **ANNOUNCE**: Display a scrolling marquee message across the top of the screen.
*   **INTRO**: Select a team in the sidebar and click "Intro" to show their colors and chant.

---

## ⌨️ 3. Power-User Shortcuts
Master these keyboard shortcuts to run the event like a pro:

| Key | Action |
| :--- | :--- |
| **1 - 5** | Switch between Setup, Host, Scoreboard, Team, and Results |
| **Space** | **Reveal** hidden questions/items OR **Clear** active buzzers |
| **Esc** | Clear buzzers / Return to Logo mode |
| **T** | Toggle Round Timer (Start/Stop) |
| **L / B / S** | Switch Projector to Logo, Blackout, or Scoreboard |
| **A** | Toggle Scrolling Announcement |
| **Digit 1-8** | Manually trigger a "Buzz In" for the corresponding team |
| **→ (Right)** | Advance to the next question/item |

---

## 🏆 4. Results & Tiebreakers
When the competition ends, click **"End Competition"** to head to the Results view.

1.  **The Reveal**: Click "Reveal Standings" to count down from last place to the champion with animations and confetti.
2.  **Tiebreakers**: If teams are tied for 1st, a **"Sudden Death"** button will appear. This opens a special buzzer showdown interface to decide the winner.
3.  **Exports**: 
    *   **Manage Certificates**: Jump to the certificate generator to print awards for participants.
    *   **Export PDF**: Generate a professional "Official Competition Report" for your records.

---

## 💡 Pro-Tips for Hosts
*   **The Confidence Monitor**: Use the "Eye" icon in the Host Panel to see a mini version of exactly what is on the projector.
*   **Real-time Sync**: If a team device loses connection, just have them refresh the page and enter the session code again; their score and state will be restored automatically.
*   **Audio**: Ensure your host laptop is connected to the venue's sound system for the built-in buzzer and reveal sound effects.
